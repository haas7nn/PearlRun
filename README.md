<h2>
  <img src="https://media.giphy.com/media/iY8CRBdQXODJSCERIr/giphy.gif" width="30"/>
  <samp> About The Game </samp>
</h2>

<table>
<tr>
<td>

🏃 **Awal** is a young Bahraini running late for a traditional **pearl diving competition**. What starts as a simple rush quickly escalates into a wild chase across the entire country as he gets caught up in chaotic situations at each location.

<br/>

<details>
<summary><b>🎯 Core Gameplay</b></summary>
<br/>

```
┌──────────────────────────────────────────────────┐
│  🏃 Run    →  Sprint through iconic locations    │
│  💎 Collect→  Gather pearls scattered across     │
│  ⚡ Dodge  →  Avoid obstacles & enemies          │
│  🎭 React  →  Quick decisions to survive         │
└──────────────────────────────────────────────────┘
```

</details>

<details>
<summary><b>🇧🇭 Cultural Experience</b></summary>
<br/>

The gameplay is filled with:
- **Bahraini cultural references**
- **Local humor**
- **Recognizable landmarks**
- **Authentic environments**

</details>

</td>
</tr>
</table>
